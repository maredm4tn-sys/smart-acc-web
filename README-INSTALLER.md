# دليل إنشاء ملف التسطيب - Smart Accountant

## الخطوات المطلوبة:

### 1. تحميل وتثبيت Inno Setup
- روح على الموقع الرسمي: https://jrsoftware.org/isdl.php
- حمل النسخة الأخيرة من **Inno Setup** (مجاني تماماً)
- سطب البرنامج على جهازك

### 2. بناء البرنامج
```bash
npm run dist
```
هذا الأمر سيقوم بـ:
- بناء تطبيق Next.js
- تجهيز ملفات Electron
- إنشاء مجلد `dist/Smart-Accountant/win-unpacked` بكل الملفات المطلوبة

### 3. إنشاء ملف التسطيب
بعد ما يخلص البناء:
1. افتح ملف `installer-setup.iss` (موجود في المجلد الرئيسي)
2. اضغط كليك يمين واختار **Compile** (أو افتحه ببرنامج Inno Setup واضغط Compile)
3. انتظر لحد ما يخلص (ثواني معدودة)

### 4. النتيجة النهائية
هتلاقي ملف التسطيب جاهز في:
```
dist/Smart-Accountant-Setup.exe
```

## ملاحظات مهمة:
- حجم ملف التسطيب هيكون حوالي 230-250 ميجا (مضغوط)
- الملف ده portable وممكن توزعه على أي جهاز Windows
- التسطيب بيحط البرنامج في `Program Files` تلقائياً
- بيعمل اختصار على سطح المكتب وفي قائمة Start

## في حالة حدوث مشاكل:
- تأكد إن Inno Setup مثبت صح
- تأكد إن مجلد `dist/Smart-Accountant/win-unpacked` موجود وفيه الملفات
- لو في مشكلة في الأيقونة، ممكن تشيل سطر `SetupIconFile` من ملف `.iss`
